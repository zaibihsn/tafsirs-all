# Quran Tafsirs Collection

This repository contains a comprehensive collection of **134 Tafsirs** (Quranic commentaries) in multiple languages. Every tafsir has been normalized to exactly **6,236 verses** to ensure compatibility across various Quranic applications.

## Key Features
- **134 Tafsirs**: Wide range of classical and modern commentaries.
- **Normalized Structure**: 114 Surah files (`1.toon` to `114.toon`) per tafsir.
- **Consistent Indexing**: Exactly 6,236 entries per repository, matching the standard Quran database.
- **CDN Optimized**: Easy to consume via jsDelivr.

## How to Access (CDN)

You can access any tafsir surah file using the jsDelivr CDN.

**URL Pattern:**
`https://cdn.jsdelivr.net/gh/zaibihsn/tafsirs-all@master/{slug}/{surah_number}.toon`

**Example (English Tafheem-ul-Quran, Surah 1):**
`https://cdn.jsdelivr.net/gh/zaibihsn/tafsirs-all@master/english-tafheem-ul-quran/1.toon`
